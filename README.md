# J.P. Morgan Quantitative Research Virtual Experience

This repository contains solutions developed during the J.P. Morgan Quantitative Research Virtual Experience Program on Forage.

---

## Project Overview

The project focuses on quantitative research techniques used in financial services, including:

- Natural gas storage contract pricing
- Credit risk modeling
- Probability of default estimation
- FICO score bucketing using dynamic programming

## Tasks Completed

### Task 1 — Data Analysis
Performed exploratory data analysis and visualization on natural gas pricing data using Jupyter Notebook.

### Task 2 — Gas Storage Pricing Model
Built a pricing engine for natural gas storage contracts considering:
- Injection schedules
- Withdrawal schedules
- Storage capacity
- Transport and storage fees

### Task 3 — Credit Risk Modeling
Developed a logistic regression model to:
- Predict probability of default (PD)
- Estimate expected loss (EL)

### Task 4 — FICO Score Bucketing
Implemented:
- MSE-based bucketization
- Log-likelihood bucketization
- Dynamic programming optimization
